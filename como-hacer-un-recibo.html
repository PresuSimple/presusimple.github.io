Cómo hacer un recibo de pago

Un recibo es un documento que confirma que se recibió un pago. Es muy utilizado por profesionales independientes y pequeños negocios.

Pasos para hacer un recibo

  1. Escribir el nombre de quien recibe el pago
  2. Agregar el nombre del cliente
  3. Indicar el monto recibido
  4. Escribir el concepto del pago
  5. Agregar la fecha
  6. Firmar el documento
